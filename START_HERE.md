## ? TROUBLESHOOTING

### No output after clicking Run
```
1. Check browser console (F12)
2. Make sure code is in editor
3. Check language selection
4. Try simpler code
5. Check internet connection
```

### Error message appears
```
1. Check code syntax
2. Try different code
3. Check browser console
4. Try test page instead
```

### Nothing happens at all
```
1. Make sure you clicked the right button (green "Run Code")
2. Wait a moment
3. Try again
4. Check F12 console for errors
```
