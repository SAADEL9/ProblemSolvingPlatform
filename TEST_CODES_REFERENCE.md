## ?? HOW TO USE THESE CODES

1. **Open test page:** http://localhost:5000/onecompiler-test.html

2. **Select language** from dropdown that matches the code

3. **Copy a code** from above

4. **Paste** into the Code textarea (big text box on left)

5. **Click** "Run Code (Direct API)" button

6. **See output** on the right side

- Error appears ? Check syntax or try simpler code
