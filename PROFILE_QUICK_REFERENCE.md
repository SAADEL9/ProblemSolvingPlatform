## ?? Change Password

### Requirements
- At least 8 characters
- Include uppercase letters (A-Z)
- Include lowercase letters (a-z)
- Include numbers (0-9)
- Include special characters (!@#$%^&*)

### Features
- **Strength Indicator** - Visual progress bar
- **Toggle Visibility** - Show/hide password
- **Requirements Display** - Clear password rules
- **Tips** - Best practices for passwords

### Steps
1. Enter current password
2. Enter new password
3. Confirm new password
4. Click "Change Password"
5. Redirected to profile on success

## ?? Common Tasks

### Task: Update Your Name
```
1. Click username dropdown
2. Select "Edit Profile"
3. Enter new first/last name
4. Click "Save Changes"
5. See success message
6. Redirected to Profile
```

### Task: Change Phone Number
```
1. Go to Edit Profile page
2. Enter phone number
3. Click "Save Changes"
4. Phone number updated
```

### Task: Change Password
```
1. Click "Change Password" (anywhere accessible)
2. Enter current password
3. Enter new password (watch strength indicator)
4. Confirm new password
5. Click "Change Password"
6. Success message shown
7. Redirected to Profile
```

### Task: View Submissions
```
1. On Profile page
2. Click "Submissions" tab
3. See table of all submissions
4. Click "View" to see submission details
```

### Task: Delete Account
```
1. Go to Edit Profile
2. Scroll to "Danger Zone"
3. Click "Delete Account"
4. Confirm in modal
5. Account deleted (irreversible)
6. Logged out automatically
```

## ?? Important Notes

### Account Deletion
?? **Warning:** Deleting your account will:
- Remove all submissions
- Remove all comments
- Remove ranking data
- **Cannot be undone**
- Will log you out automatically

### Password Change
? **Note:** After changing password:
- You stay logged in
- Session is refreshed
- Use new password for future logins

### Email Changes
? **Note:** Email cannot be changed from profile
- Contact admin if email needs change
- Security measure to prevent unauthorized changes
