## ? IF NOT WORKING

### Check 1: Code in Editor?
- Should see: `print("Hello World!")`
- If empty: Paste code again

### Check 2: Language Selected?
- Dropdown should show: Python
- If different: Select Python

### Check 3: Button Clicked?
- Make sure you clicked green "Run Code" button
- Not "Submit" or "Clear"

### Check 4: Open Console
- Press F12
- Go to Console tab
- Click "Run Code" again
- Look for error messages

### Check 5: Check Internet
- Can you visit https://onecompiler.com?
- If no: Network issue
- If yes: Try the test page
